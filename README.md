## barber-shop

### Description
This is training version Sinatra app Barber Shop without Active Record

Language: Ruby, version: 2.5.3

DataBase: SQLite3

Gem Sinatra

### To start
* Clone **barber_shop**

* Use bundle
```
$ bundler install
```
* In command line type
```
$ ruby app.rb
```
* Open browser with URL
```
localhost://4567
```
